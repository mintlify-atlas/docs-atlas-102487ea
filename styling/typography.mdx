# Typography System

MYMUSICK uses distinct typography strategies in each theme. The original theme features a custom Gyanko font, while the formal theme uses modern Google Fonts with a clear hierarchy.

## Font Families

<Tabs>
  <Tab title="Original Theme">
    ### Gyanko Custom Font

    The original theme's primary brand font is **Gyanko**, a custom typeface loaded via `@font-face`.

    <CodeGroup>
    ```css Font Declaration
    @font-face {
      font-family: 'Gyanko';
      src: url("fonts/gyanko-regular.otf") format("opentype");
      font-weight: normal;
      font-style: normal;
    }
    ```

    ```css Base Typography
    body {
      font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
      color: var(--text-light);
    }
    ```
    </CodeGroup>

    ### Font Stack

    | Element | Font Family | Purpose |
    |---------|-------------|----------|
    | Body | `'Segoe UI', Tahoma, Geneva, Verdana, sans-serif` | Body text, metadata |
    | `.title` | `'Gyanko', sans-serif` | Logo/brand text |
    | `.bttlg` | `'Gyanko', sans-serif` | Buttons |
    | `.sidebar a` | `'Gyanko', sans-serif` | Navigation links |
    | `.intxt` | `'Gyanko', sans-serif` | Section headings |

    ### Character Set

    <Note>
      Gyanko is a display font best used for headings and UI elements. Ensure the font file is available at `fonts/gyanko-regular.otf` relative to your CSS file.
    </Note>

    ### Usage Examples

    <CodeGroup>
    ```css Title / Logo
    .title {
      font-family: 'Gyanko', sans-serif;
      font-size: 40px;
      letter-spacing: 2px;
      color: var(--accent);          /* #FF5757 */
      font-weight: normal;
      margin: 0;
    }
    ```

    ```css Navigation
    .sidebar a {
      font-family: 'Gyanko', sans-serif;
      font-size: 24px;               /* Large, readable */
      color: white;
      text-decoration: none;
    }
    ```

    ```css Section Headings
    .intxt {
      margin-top: 40px;
      font-family: 'Gyanko', sans-serif;
      font-size: 25px;
      text-align: center;
    }
    ```

    ```css Buttons
    .bttlg {
      font-family: 'Gyanko', sans-serif;
      font-size: 16px;
      /* Bold, custom appearance */
    }
    ```
    </CodeGroup>
  </Tab>

  <Tab title="Formal Theme">
    ### Google Fonts Stack

    The formal theme combines **Bebas Neue** (display) with **DM Sans** (body) for a professional, modern aesthetic.

    <CodeGroup>
    ```css Font Import
    @import url('https://fonts.googleapis.com/css2?family=Bebas+Neue&family=DM+Sans:ital,wght@0,300;0,400;0,500;1,300&display=swap');
    ```

    ```css CSS Variables
    :root {
      --font-display: 'Bebas Neue', sans-serif;
      --font-body:    'DM Sans', sans-serif;
    }
    ```

    ```css Base Typography
    body {
      font-family: var(--font-body);     /* DM Sans */
      font-size: 15px;
      color: var(--text);                /* #e8e6df */
    }
    ```
    </CodeGroup>

    ### Font Roles

    | Font | Variable | Usage | Characteristics |
    |------|----------|-------|----------------|
    | **Bebas Neue** | `--font-display` | Titles, headings, large text | All-caps, condensed, bold |
    | **DM Sans** | `--font-body` | Body, UI text, labels | Clean, geometric, legible |

    ### DM Sans Weight Scale

    <CardGroup cols={4}>
      <Card title="Light: 300" icon="feather">
        Metadata, captions
      </Card>
      <Card title="Regular: 400" icon="align-left">
        Body text
      </Card>
      <Card title="Medium: 500" icon="bold">
        Buttons, labels
      </Card>
      <Card title="Light Italic: 300" icon="italic">
        Emphasis
      </Card>
    </CardGroup>

    ### Usage Examples

    <CodeGroup>
    ```css Display: Logo
    .title {
      font-family: var(--font-display);  /* Bebas Neue */

      letter-spacing: 3px;
    ```

    ```css Display: Section Heading

    ```

    ```css Display: Dialog Heading

      font-family: var(--font-display);  /* Bebas Neue */
      font-size: 32px;
      letter-spacing: 2px;
    ```

    ```css Body: Navigation

      font-family: var(--font-body);     /* DM Sans */

      letter-spacing: .2px;
    ```

## Letter Spacing

Both themes use letter-spacing for display typography to enhance readability and brand presence.

<CodeGroup>
```css Original Theme
.title {
  letter-spacing: 2px;     /* Logo: moderate tracking */
}

/* No letter-spacing on other elements */
```

```css Formal Theme
.title {
  letter-spacing: 3px;     /* Logo: wider tracking */
}

.intxt {
  letter-spacing: 1px;     /* Hero: subtle tracking */
}

.login-form h3 {
  letter-spacing: 2px;     /* Dialog: balanced */
}

.sidebar a {
  letter-spacing: .2px;    /* Nav: micro adjustment */
}

.bttlg {
  letter-spacing: .5px;    /* Button: slight spacing */
}
```
</CodeGroup>

<Note>
  Bebas Neue naturally has condensed proportions, so increased letter-spacing improves legibility at display sizes.
</Note>

## Font Loading Strategy

<Tabs>
  <Tab title="Original: @font-face">
    ### Self-Hosted Font

    <Steps>
      <Step title="Font Declaration">
        ```css
        @font-face {
          font-family: 'Gyanko';
          src: url("fonts/gyanko-regular.otf") format("opentype");
          font-weight: normal;
          font-style: normal;
        }
        ```
      </Step>

      <Step title="File Structure">
        Ensure the font file exists at:
        ```
        project/
        ├── css/
        │   └── estilooriginal.css
        └── fonts/
            └── gyanko-regular.otf
        ```
      </Step>

      <Step title="Fallback">
        Always include fallback fonts:
        ```css
        font-family: 'Gyanko', sans-serif;
        ```
      </Step>
    </Steps>

    <Warning>
      If `gyanko-regular.otf` fails to load, the browser will fall back to the default sans-serif font, which may significantly change the appearance.
    </Warning>
  </Tab>

  <Tab title="Formal: Google Fonts">
    ### CDN-Hosted Fonts

    <Steps>
      <Step title="Import Statement">
        ```css
        @import url('https://fonts.googleapis.com/css2?family=Bebas+Neue&family=DM+Sans:ital,wght@0,300;0,400;0,500;1,300&display=swap');
        ```
      </Step>

      <Step title="CSS Variables">
        ```css
        :root {
          --font-display: 'Bebas Neue', sans-serif;
          --font-body:    'DM Sans', sans-serif;
        }
        ```
      </Step>

      <Step title="Apply Variables">
        ```css
        body { font-family: var(--font-body); }
        .title { font-family: var(--font-display); }
        ```
      </Step>
    </Steps>

    ### Loading Optimization

    The `display=swap` parameter in the Google Fonts URL ensures:
    - **Immediate text rendering** with fallback fonts
    - **Swap to web fonts** once loaded (no invisible text)
    - **Better perceived performance**

    <Note>
      Google Fonts are cached across domains, so users likely already have Bebas Neue and DM Sans loaded from other sites.
    </Note>
  </Tab>
</Tabs>

## Accessibility

### Font Size Minimums

<Warning>
  Ensure body text is at least 14px for readability. Both themes meet this requirement:
  - Original: 16px (search), 14px+ (song info)
  - Formal: 15px (body), 13px+ (UI elements)
</Warning>

### Relative Units

Consider using `rem` instead of `px` for better accessibility:

```css
/* Current (px) */
body { font-size: 15px; }

/* Accessible (rem) */
html { font-size: 100%; }  /* 16px default */
body { font-size: 0.9375rem; }  /* 15px at default zoom */
```

<Note>
  Using `rem` units allows users to scale text globally via browser settings without breaking your layout.
</Note>
